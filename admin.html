对应前台 #scr-style，管理前台展示的风格卡片
info页面结构说明
touch_app
前台风格页顶部有分类筛选栏（由风格的 category 字段自动生成）， 下方是 2列风格卡片网格。每张卡片包含：封面图、风格名称、NEW 徽章（排序最前的）。 下方"风格库管理"可增删改所有风格。
lightbulb
小技巧：分类标签填写相同的字符串（如「梦幻风」「复古风」），前台筛选栏会自动汇总并显示。 排序数字越小越靠前，排序最靠前的风格会显示「NEW」徽章。

待机吸引页
对应前台 #scr-attract，设备无操作时展示
info页面结构说明
info
待机吸引页结构：
1. 全屏背景图/视频（.attract-bg）— 目前使用 picsum 占位图，建议替换为真实品牌图
2. Logo + 标题（.attract-title）
3. 点击开始按钮（.attract-tap-ring）
4. 底部滚动风格标签（.attract-tags-inner）— 自动从风格库读取风格名称
construction
待机页背景图在前台代码 potoca_v9_fixed.html 中 #scr-attract 区域的 background-image 样式替换。底部标签文字由风格库的 name 字段自动生成， 管理风格名称即可自动更新标签。

结果页
对应前台 #scr-result，AI 生成后的展示页
info页面结构说明
info
结果页包含：
1. 大图主显示（#result-main-img）— 由 AI 生成，运行时动态填充
2. 4张缩略图选择（.result-thumb）— 对应 JS 变量 resultPhotos[]
3. 风格名称标签（#result-style-label）— 自动显示已选风格名
4. 底部操作栏：打印、分享按钮
construction
结果页的缩略图在前台 JS 的 resultPhotos 数组中配置（约 4909 行附近）， 修改数组中的图片 URL 即可更换结果页示例图。

⚠️ 如果新增风格报错 "row-level security policy"，需要在 Supabase 后台开启写入权限：

1. 打开 supabase.com → 你的项目
2. 左侧 Table Editor → styles 表 → 右上角 RLS
3. 点 Add Policy → 选 Enable insert/update/delete for all users
4. 左侧 Storage → style-images bucket → Policies
5. 添加 INSERT 权限给 anon 角色
info 配置修改后请同步更新 potoca_v9_fixed.html 中的 SUPABASE_URL 和 SUPABASE_ANON_KEY 变量。
